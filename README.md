# Password Manager

A simple and secure password manager built using Python and Tkinter. This GUI application allows you to generate strong passwords, store them securely in a local JSON file, and retrieve them easily.

## Preview

[Screenshot](screenshot.png) 


## Features

- Generate random, secure passwords
- Store website credentials (website, email, password)
- Retrieve saved credentials by searching the website
- Copy generated passwords to clipboard automatically
- Friendly GUI built with Tkinter
- JSON file storage for persistence


## Tech Stack

- **Python 3**
- **Tkinter** (for GUI)
- **JSON** (for storing credentials)
- **pyperclip** (for clipboard support)
- **random** (to generate passwords)
